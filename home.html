My Fabulous Cactus Shop

The goal of this assignment was to perform an A/B test, collect data and conduct a statistical analysis to gather insight on two different UI designs. I have included a link to the webpages I used to conduct the A/B test above. For these pages, I made relatively minor UI changes, mostly stylistic and mostly dealing with the visual appearance and organization of the page.s Versison A was the edited page while version B was the original template given in class.

THE REASONING
  • My alternate hypothesis for both metrics is informed by the changes I made to the HTML layout.
  • In Version B, everything wasnâ€™t centered which made flow and overall presentation not as clear and fluid.
  • In Version A, the cacti images, add to cart and shopping cart features were all centered on the page.
  • Additionally, there was a difference between the title and the content of the page in terms of color.
  • This delineation of content makes it more clear to the user which section they will most likely be completing tasks in (I think).
  • As a result, if tasks they are meant to complete are easier to see in Version A, time to completion will be shorter and there will be a difference in return rate (in favor of Version A).

    • THE HYPOTHESIS
    Time to Completion:
  • Null: There is no difference in time to completion between Version A and Version B (Version A is not better than Version B).
  • Alternative: Version A is better than Version B ( it will take less time to complete task).
    • Return Rate:
  • Null: There is no difference in return rate between Version A and Version B.
  • Alternative: Version A is different from Version B (different return rate).
    • Data Collection Process
  • As outlined in class, the link was sent out to classmates.
  • However, I didn't have nearly enough data points (nowehere near 40), so I sent my link to friends and classmates from other classes.
  • Regardless of which version of the pages a user received, their task was to fill their cart with at least $150 worth of cacti/succulents

    • THE RESULTS
    Time to Completion:
  • The t-score between the two sample means for TIME TO COMPLETION: 0.03621766934688187
  • The p-value for the test of TIME TO COMPLETION: 0.48567050784590693
    • Return Rate:
  • the chi^2 value for the test of RETURN RATE: 0.22490530303030304
  • the p-value for the test of RETURN RATE: 0.6353274751385691
    • THE CONCLUSIONS
    Time to Completion:
  • Since my p-value > significance level (.05), I fail to reject the null hypothesis.
  • That is, I fail to reject that there was no difference in time to completion between Version A and Version B of my pages.
  • Overall, my results were statistically insignificant giving doubt as to whether my alternative hypothesis will repeat in future tests.
    • Return Rate:
  • Again, since my p-value > significance level (.05), I fail to reject the null hypothesis.
  • That is, I fail to reject that there was no difference in return rate between Version A and Version B of my pages.
